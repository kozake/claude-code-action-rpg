# BGM ファイル配置

このディレクトリに以下の音声ファイルを配置してください。

## 必要なファイル

| ファイル名 | 用途 | ダウンロード元 |
|-----------|------|---------------|
| `field.ogg` / `field.mp3` | フィールド探索BGM | [JRPG Pack 1 Exploration](https://opengameart.org/content/jrpg-pack-1-exploration) より "Grasslands" などを使用 |
| `boss.ogg` / `boss.mp3` | ボス戦BGM | [Intense Boss Battle](https://opengameart.org/comment/111766) を使用 |

## ライセンス

すべて **CC0（パブリックドメイン）** ライセンスです。
著作権表示不要で自由に使用できます。

## 備考

- OGG形式を優先し、MP3をfallbackとして使用します
- ファイルが存在しない場合はエラーを出さずに無音で動作します
